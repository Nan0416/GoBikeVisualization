GoBike Data Visualization

A tool helps efficiently distribute bikes across stations.

The GoBike Data Visualization project was a final project of Gatech CS 7260 Information Visualization. It was developed by Nan Qin, Jiawei Zhang, and Wenyi Qiu. It provided an efficient tool to help the GoBike management team to learn the pattern of bike usage. Therefore, helping maximum bike usage by efficiently distributing bikes. This project visiualized bike usage by weekly and monthly.
